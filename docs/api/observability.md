# `bca2p.observability`

`bca2p.observability` provides structured runtime events, trace recording, JSON export, topology snapshots, replay bundles, and diagnostic helpers.

## Main Objects

- `TraceRecorder`
- `RuntimeEvent`
- `RuntimeEventType`
- `TopologySnapshot`
- `ReplayEventBundle`
- `VisualizationGraph`

## Diagnostics

- `detect_bottlenecks(...)`
- `noisy_sender_report(...)`
- `dormant_receptor_report(...)`
- `amplification_storm_report(...)`
