# `bca2p.learning`

`bca2p.learning` provides the stable causal feedback and policy refinement layer.

## Main Objects

- `CausalGraphStore`
- `CausalSignalEdge`
- `CausalDecisionNode`
- `CausalOutcomeNode`
- `ContributionSummary`
- `CounterfactualResult`
- `PolicyRefinementProposal`

## Supported Stable Behaviors

- causal signal recording
- decision recording
- feedback ingestion
- contribution summaries
- counterfactual queries
- safe topology/amplification/complex refinement proposals
- deterministic fallback routing selection
